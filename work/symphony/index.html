Symphony

Web App

Discovering new music should be easy.

Tell Symphony about a musician you like and it'll suggest others. I worked on Symphony for the 24-hour Yahoo! Hack U 2009 hackathon. Symphony was both my first ever web app project, and it ignited my interest in web development. It was also a solo project, so it presented many challenges and lessons.

Technology

Symphony uses the last.fm API to gather data about related artists. Working on Symphony was my first time using JavaScript and jQuery, and I learned the basics on the spot for the hackathon.

Design

Symphony presents musicians in a tiled photo view. Clicking on a tile opens more detailed information about the musician in a modal windows. This information includes a biography and larger image. I also originally intended to display information like YouTube videos and Flickr photos but ran out of time.
